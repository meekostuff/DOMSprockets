XBL

A Javascript library providing a DOM binding / behavior engine.

Background

XBL stands for XML Binding Language. This code DOES NOT support currently support XML or XBL, but it is derived from the XBLUI project which is a (partial) Javascript implementation of the initial XBL 2.0 specification.

The name is not so appropriate now because XBL2 is no longer recommended and XML is out of favor on the web.

The code now supports a Javascript API similar to that envisaged by the XBL 2.0 working group note of 2012.
